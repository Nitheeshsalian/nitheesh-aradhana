# nitheesh-aradhana
 Wedding App

## Overview
A wedding planning and management application designed to help couples and their families organize every detail of their special day.

## Features
- Guest list management
- RSVP tracking
- Budget planning
- Timeline and checklist
- Vendor management
- Seating arrangements

## Installation
```bash
npm install
```

## Usage
```bash
npm start
```

## Technologies
- Node.js
- React
- MongoDB
